Installation

To use r128x, you just have to copy the r128x executable file somewhere on your hard drive.

You could for example copy it in the /usr/local/bin/ folder :


Mount the r128x_0_1.dmg image by double-clicking it.

Launch Terminal.app.

In the Terminal, execute the following command (warning : if there already is a r128x file in the /usr/local/bin/ directory, it will be overwritten without warning) :

sudo cp /Volumes/r128x_0_1/r128x /usr/local/bin/

The Terminal will ask for an admin password. Type your admin password (which is not displayed on screen) and press return.


You can check that the copy was successful by executing the following command in the Terminal :

which r128x

which should return

/usr/local/bin/r128x


Usage

r128x will try to process all files passed as arguments when invoked from the Terminal.

Tip : You can drag and drop trom the Finder to the Terminal to add files to be processed.


Example 1 : processing one file.


Macintosh:~ manunaudin$ r128x /Users/manunaudin/Documents/audio/loud_test_files/ebu-loudness-test-setv03/1kHz\ Sine\ -20\ LUFS-16bit.wav 

/Users/manunaudin/Documents/audio/loud_test_files/ebu-loudness-test-setv03/1kHz Sine -20 LUFS-16bit.wav -20.0 0.0 -19.9

Macintosh:~ manunaudin$


The second line is the output of r128x indicating the full file path, the Integrated Loudness in LUFS, the Loudness Range in LU and the Maximum True Peak in dBTP.

If a file cannot be processed, you will get a (not very detailed) error message.


Example 2 : trying to process a file that is not supported.


Macintosh:~ manunaudin$ r128x /Users/manunaudin/Rplots.pdf 

An error occured while trying to open/read/process file /Users/manunaudin/Rplots.pdf

Macintosh:~ manunaudin$ 


You can process several files at a time by passing them as separate arguments.


Example 3 : processing several files.


Macintosh:~ manunaudin$ r128x /Users/manunaudin/Documents/audio/1063721.wav /Users/manunaudin/Documents/audio/1063811.wav 

/Users/manunaudin/Documents/audio/1063721.wav -21.1 1.8 -9.3

/Users/manunaudin/Documents/audio/1063811.wav -22.0 10.3 -13.0

Macintosh:~ manunaudin$ 


You can process all/several files in a directory by using file name expansion.
