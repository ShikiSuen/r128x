Installation

Pour utiliser r128x, vous devez simplement copier le fichier exécutable r128x sur votre disque dur.

Vous pouvez par exemple le copier dans le répertoire /usr/local/bin/ :


Montez l'image disque r128x_0_1.dmg en double-cliquant dessus.

Lancez l'application Terminal.app.

Dans le Terminal, exécutez la commande suivante (Attention : s'il il y a déjà un fichier nommé r128x dans le répertoire /usr/local/bin/, il sera remplacé sans avertissement) ;

sudo cp /Volumes/r128x_0_1/r128x /usr/local/bin/

Le Terminal vous demande un mot de passe administrateur. Saisissez votre mot de passe administrateur (qui n'est pas affiché) et pressez retour pour valider.


Vous pouvez vérifier que la copie est bien effectuée en exécutant la commande suivante dans le Terminal :

which r128x

qui devrait afficher :

/usr/local/bin/r128x


Utilisation

r128x essayera d'analyser tous les fichiers passés en arguments lorsque vous l'invoquez depuis le Terminal.

Truc : vous pouvez glisser/déposer du Finder vers le Terminal pour ajouter des fichiers à analyser. 


Exemple 1 : analyser un fichier.


Macintosh:~ manunaudin$ r128x /Users/manunaudin/Documents/audio/loud_test_files/ebu-loudness-test-setv03/1kHz\ Sine\ -20\ LUFS-16bit.wav 

/Users/manunaudin/Documents/audio/loud_test_files/ebu-loudness-test-setv03/1kHz Sine -20 LUFS-16bit.wav -20.0 0.0 -19.9

Macintosh:~ manunaudin$


La seconde ligne est le résultat de l'analyse et indique la chemin complet du fichier, le Loudness Intégré en LUFS, le Loudness Range en LU et le Maximum True Peak en dBTP.

Si un fichier n'a pas pu être analysé, vous obtiendrez un message d'erreur.


Exemple 2 : essai d'analyse d'un fichier non pris en charge.


Macintosh:~ manunaudin$ r128x /Users/manunaudin/Rplots.pdf 

An error occured while trying to open/read/process file /Users/manunaudin/Rplots.pdf

Macintosh:~ manunaudin$ 


Vous pouvez analyser plusieurs fichiers à la fois en les passant comme arguments successifs


Exemple 3 : analyse de plusieurs fichiers.


Macintosh:~ manunaudin$ r128x /Users/manunaudin/Documents/audio/1063721.wav /Users/manunaudin/Documents/audio/1063811.wav 

/Users/manunaudin/Documents/audio/1063721.wav -21.1 1.8 -9.3

/Users/manunaudin/Documents/audio/1063811.wav -22.0 10.3 -13.0

Macintosh:~ manunaudin$ 


Vous pouvez analyser plusieurs/tous les fichiers d'un répertoire en utilisant les jockers.
