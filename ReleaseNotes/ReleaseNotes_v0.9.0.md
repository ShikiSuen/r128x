# Release Notes - r128x v0.9.0

Released: September 2025

## English

### Quick Audition Around the First dBTP Location

r128x v0.9.0 allows users to quick-audition around the first dBTP location. 5 more UI languages are supported.

- [macOS] You can now also use the Mac App Store version of this app as CLI app. Run `/Applications/r128x-swiftui.app/Contents/MacOS/r128x-swiftui --cli --help` to see available options.
- Added new UI languages: Deutsch, Spanish, Brazilian (pt-BR), Russian, and Turkish.
- Since this release, this app allows you to immediately audition the 3-second part around the first dBTP location by tapping / clicking a measured track. You may also use the up / down arrow key on your keyboard to navigate among measured tracks and hit the space key to do the same audition action.
- There is a lack of evidence regarding which macOS / iOS annual release introduced RF64 support. Therefore, in this release, this app introduced failure management procedures for loudness measurements against RF64 audio formats. At least, we have tested on OS26 that RF64 audio files are measurable through this app.
